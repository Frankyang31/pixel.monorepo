# 博客列表页 设计说明

**原型文件**：`blog-design.html`  
**主题**：Light Theme  
**渲染策略**：SSG（内容型页面，SEO 优先，构建时生成）  
**路由**：`/blog`  
**最后更新**：2026-03-24

---

## 设计目标

博客列表页承担内容营销和 SEO 引流的核心职责，目标是**吸引通过搜索引擎到来的内容型用户，建立品牌的专业形象，将内容读者转化为产品用户**。核心路径：通过 SEO/社交分享到达 → 浏览文章列表 → 阅读感兴趣的文章 → 点击文章内 CTA → 注册/体验工具。

---

## 主题策略

Light Theme，原因：
1. 博客属于公域内容，面向搜索引擎和新用户，浅色风格阅读体验更佳。
2. 与首页视觉一致，降低用户跳转后的认知差异。
3. 文字阅读类内容在浅色背景下清晰度高于深色（长篇内容）。

---

## 区块结构与设计决策

### 1. 顶部导航（公域版）

**内容**：Logo + 主导航 + 登录/注册按钮。

**与设计规范关联**：`--bg-base`、`--border-subtle`

---

### 2. 博客 Hero 区

**内容**：左侧大标题 + 描述文字，右侧精选文章大图卡（焦点文章）。

**设计决策**：
- Hero 采用左右分割布局而非全宽背景图，右侧精选文章为强引导入口。
- 精选文章卡使用渐变叠层图片 + 大号标题，视觉冲击力强。
- Hero 高度保持适中，文章列表尽快进入视野。

**与设计规范关联**：`--brand-start`、`--brand-end`、`--radius-2xl`

---

### 3. 分类 Tab 过滤栏

**内容**：全部 / 教程 / 产品更新 / AI 资讯 / 创作技巧 / 行业洞察，横向排列。

**设计决策**：
- Tab 而非下拉，分类数量适中（6~8 个），一键切换成本最低。
- Tab 保持 `sticky` 在导航下方，方便长页面中切换分类。

**与设计规范关联**：`--radius-full`、`--border-brand`

---

### 4. 文章列表主区

**内容**：三列文章卡（宽屏），每张卡含封面图、分类标签、标题、摘要、作者头像+姓名、发布日期、阅读时长。

**设计决策**：
- 三列网格适合内容展示，兼顾信息密度与视觉舒适度。
- 每张卡的封面图使用固定高宽比（16:9），避免参差不齐影响整齐感。
- 阅读时长标注（"5 min read"）是 Medium 风格设计，帮助用户快速决策是否阅读。
- 文章卡 hover 时标题下划线 + 轻微升起动效，提示可点击。

**与设计规范关联**：`--bg-surface`、`--shadow-sm`、`--radius-xl`

---

### 5. 侧边栏（可选）

**内容**：热门文章 Top 5 + 订阅邮件列表入口 + 分类快速导航。

**设计决策**：
- 侧边栏适合内容较多时辅助导航，MVP 阶段可暂不实现，列表页保持纯三列网格。
- 邮件订阅是低成本的用户触达渠道，适合内容型产品冷启动阶段。

---

### 6. 分页控件

**内容**：数字分页 + 上/下一页按钮。

**设计决策**：
- 博客内容适合分页而非无限滚动，便于 SSG 生成固定页面以利 SEO。

---

## 交互细节

| 交互 | 行为 | 说明 |
|------|------|------|
| 点击分类 Tab | 切换文章列表 | 路由切换 or 客户端过滤 |
| 文章卡 hover | 标题出现下划线，卡片轻微升起 | CSS transition |
| 点击文章卡 | 跳转到博客详情页 `/blog/[slug]` | 页面跳转 |
| 点击作者头像/名 | 跳转作者主页（规划功能） | 暂 disabled |
| 订阅邮件（侧边栏） | 输入邮箱提交后显示确认信息 | 表单提交 |

## 响应式策略

| 断点 | 布局变化 |
|------|----------|
| ≥1280px | 3 列文章卡 |
| 1024-1279px | 3 列（间距略减） |
| 768-1023px | 2 列 |
| <768px | 1 列，Hero 单列，封面图全宽 |

---

## 可访问性说明

- 文章卡整体为可点击区域，使用 `<a>` 包裹 `<article>` 结构。
- 分类 Tab 使用 `role="tablist"` 和 `aria-selected`。
- 封面图 `alt` 属性包含文章标题。
- 色彩对比：文章标题（`--text-primary`）on `--bg-surface` ≥ 4.5:1（AA）。
