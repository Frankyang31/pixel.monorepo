# 作品广场 设计说明

**原型文件**：`gallery-design.html`  
**主题**：Dark Theme  
**渲染策略**：SSR（需实时数据，兼顾 SEO）  
**路由**：`/gallery`  
**最后更新**：2026-03-24

---

## 设计目标

作品广场是平台的内容社区入口，核心目标是**激发用户创作欲望、建立社区归属感、驱动新用户注册转化**。目标用户：①已登录用户（分享、探索灵感）；②未登录访客（被内容吸引，注册转化）。核心路径：浏览作品 → 发现感兴趣的 → 点击查看详情 → （未登录）注册 → 尝试生成。

---

## 主题策略

采用 Dark Theme，原因：
1. AI 生成的图像和视频在深色背景下视觉表现最佳，颜色更鲜艳，对比度更高。
2. 沉浸式深色界面营造"创意工作室"氛围，与工作台保持一致的产品调性。
3. 参考 Midjourney、Leonardo.ai 等竞品，暗色广场已是行业范式。

## 区块结构与设计决策

### 1. 顶部导航（公域版）

**内容**：Logo + 主导航（工具/广场/定价/博客）+ 登录/注册按钮。

**设计决策**：
- 作品广场属于公域页面，保留完整导航链接，方便新访客探索其他页面。
- 导航背景使用毛玻璃效果（`backdrop-filter: blur`）+ 半透明，不遮挡滚动内容。

**与设计规范关联**：`--bg-base`、`--border-subtle`

---

### 2. 页头与过滤栏

**内容**：大标题 + 副标题 + 分类 Tab（全部/风景/人物/建筑等）+ 排序下拉 + 搜索框。

**设计决策**：
- 过滤栏设置 `position: sticky; top: 64px`，随页面滚动固定在导航下方，方便用户随时切换分类。
- 排序下拉（最新/最热/最多收藏）满足不同浏览需求，默认"最新"体现内容活跃感。
- 搜索框在过滤栏右侧，不单独占行，节省空间。

**与设计规范关联**：`--bg-surface`、`--border-subtle`、`--radius-full`

---

### 3. 瀑布流图片网格

**内容**：多列不等高瀑布流，每张卡片包含图片、作者头像、作者名、点赞数，悬浮显示更多操作。

**设计决策**：
- 使用 CSS `columns` 实现瀑布流而非 Masonry JS 库，减少依赖，性能更好。
- 5 列布局（宽屏）适合浏览大量内容，列间距 12px 保持紧凑。
- 悬浮遮层（overlay）包含点赞/收藏/查看按钮，保持卡片平时的干净感。
- 不等高设计让页面视觉更丰富，符合 AI 生图的多样比例特性（横/竖/方形）。

**与设计规范关联**：`--bg-surface`、`--radius-xl`、`--shadow-lg`

---

### 4. 空状态 / 加载态

**内容**：未加载时展示 Skeleton 占位动画；无匹配内容时展示友好的空状态提示。

**设计决策**：
- Skeleton 使用 CSS 动画（shimmer effect），避免空白区域造成加载感差。
- 空状态包含一个"去生成第一张"的引导按钮，将空状态转化为转化机会。

**与设计规范关联**：`--bg-elevated`、`--text-tertiary`

## 交互细节

| 交互 | 行为 | 说明 |
|------|------|------|
| 滚动至底部 | 加载下一页（无限滚动） | Intersection Observer 触发 |
| 点击分类 Tab | 切换内容过滤 | 重新请求 or 客户端过滤 |
| 悬浮图片卡 | 显示半透明遮层 + 操作按钮 | CSS hover opacity transition |
| 点击图片 | 进入作品详情页 `/gallery/[id]` | 页面跳转 |
| 点击点赞 | 点赞数 +1，图标高亮 | 需登录，否则引导注册弹窗 |
| 点击作者名 | 跳转到作者主页（规划功能） | 暂时 disabled |

## 响应式策略

| 断点 | 布局变化 |
|------|----------|
| ≥1280px | 5 列瀑布流 |
| 1024-1279px | 4 列 |
| 768-1023px | 3 列 |
| <768px | 2 列，过滤栏 Tab 横向滑动 |

---

## 可访问性说明

- 每张图片卡必须有 `alt` 属性，内容为 Prompt 前 50 个字符。
- 点赞按钮 `aria-pressed="true/false"` 反映当前状态。
- 键盘用户可通过 Tab 键在卡片间导航，Enter 键进入详情页。
- 图片加载失败时，显示占位图 + `aria-label="图片加载失败"`。
