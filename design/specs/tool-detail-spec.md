# 工具详情页（Tool Detail）— 设计说明

> 文件路径：`design/pages/tool-detail-design.html`
> 最后更新：2026-03-25
> 页面主题：Light Theme
> 当前示例工具：角色立绘（Character Art）
> 优先级：P1

---

## 一、设计目标

### 页面定位
工具详情页是 SEO 流量最重要的承接页。每个 AI 游戏工具各有独立 URL（如 `/tools/character-art`），是通过搜索引擎进来的核心落地页。

### 核心转化路径
```
开发者搜索"AI游戏角色立绘" → 工具详情页 → 理解能力边界 → 被作品展示种草 → 点击「立即创作」进工作台
```

### 成功指标
- 开发者 10 秒内理解"这个工具能帮我解决什么问题"
- 作品展示区能有效激发"我也想做一个"的冲动
- Hero CTA 点击率 > 8%
- 页面跳出率 < 50%

---

## 二、主题策略

使用 **Light Theme（浅色主题）**，原因：
- 工具详情页属于公域营销页，需要建立信任感，降低游戏开发者的使用门槛
- Light Theme 有利于 SEO 内容抓取，文字和结构更清晰
- 对比工具网格页，详情页有大量图片展示区，浅色背景能更好地衬托 AI 生成内容的色彩
- 与首页风格一致，用户从首页点击工具卡片进入，体验连贯

---

## 三、区块结构与设计决策

### 3.0 导航（Nav）
**一致性复用**：与首页完全一致的 sticky nav。面包屑导航显示当前层级：`首页 › AI 游戏工具 › 角色立绘`，帮助开发者理解页面位置，同时提升 SEO 内链结构。

---

### 3.1 Hero 区

**布局**：两列 Grid，左侧文案 + 右侧视觉，50/50 分配。

**左侧**包含：
- 功能徽章：带绿色脉冲动点，传达"工具正常运行"的实时感
- 大标题：`44px / 800 weight`，包含渐变文字强调"AI + 游戏制作"核心价值
- 描述文字：控制在 3 行以内，说明模型能力、画风支持和输出格式
- 双 CTA：主按钮「立即创作」（带积分说明，降低心理门槛）+ 次按钮「查看生成案例」
- 3 项数据：生成量（200K+）、用户满意度（98%）、生成速度（~6s）——用具体数字建立信任

**右侧视觉设计决策**：
- 使用 `2×2` 网格展示不同风格的 AI 游戏角色作品，左侧第一格 `grid-row: 1/3` 占全高，形成视觉主次层次
- 背景使用深色（`#1a1a2e`）营造"工作台内部"的透视感，与 Light Theme 页面形成对比，暗示"进去之后是专业创作空间"
- 两个悬浮信息卡：左上角"最新生成时间 + 画风标签"、右下角"今日生成量 + 增长率"

---

### 3.2 功能特点（Features）

**目的**：快速让游戏开发者理解"这个工具的差异化能力是什么"，区别于通用 AI 绘图工具。

**布局**：`3列 × 2行` 共 6 张特性卡片，白色背景 section 与 Hero 的暖灰背景形成交替节奏。

**6 项特性**（针对游戏开发场景设计）：
1. **游戏画风支持**（30+ 画风预设）——覆盖 RPG、SLG、二游等主流游戏类型
2. **透明背景 PNG**（直接导入引擎）——无需手动抠图，直接用于 Unity / Unreal / RPG Maker
3. **多视角输出**（正面 + 侧面）——角色设计标准交付物，减少返工
4. **精细参数控制**（CFG、Steps、Seed）——专业用户可精确控制每个生成变量
5. **批量 & 变体**（1-4 张/次）——快速探索不同设计方案
6. **风格一致性模板**（保存 Prompt 预设）——确保团队输出风格统一

---

### 3.3 作品展示（Gallery）

**目的**：这是最重要的种草区，用真实游戏角色作品让开发者产生"我的角色也能这么好看"的期待。

**布局**：CSS `columns: 4` 瀑布流展示不同游戏类型的角色设计。

**筛选标签**：按游戏类型筛选（全部 / RPG / 二游 / SLG / 像素游戏 / 国风）

**交互设计**：
- 默认状态：只展示图片，最大化视觉冲击
- hover 状态：渐变遮罩 + Prompt 示例 + 使用的画风标签淡入

---

### 3.4 使用流程（How It Works）

**目的**：消除"我不会用"的上手顾虑，展示用这个工具完成一个游戏角色的标准流程。

**布局**：3 步横向排列，步骤之间有水平连接线。

**3 步设计**（面向游戏开发者）：
1. **输入角色描述** — 用 Prompt 描述角色种族、职业、服装、武器，参考右侧示例
2. **选择画风与视角** — 从 30+ 画风预设中选择，支持正面 + 侧面双视角
3. **生成 & 下载** — ~6 秒出图，满意则下载 PNG，不满意一键生成变体

---

### 3.5 Prompt 模板区（新增区块，2026-03-25）

**目的**：降低游戏开发者的 Prompt 编写门槛，提供直接可用的模板。

**内容**：展示 3-4 个按游戏类型分类的 Prompt 模板卡片：
- RPG 奇幻角色模板
- 二游/抽卡角色模板
- SLG 军事角色模板
- 像素游戏角色模板

**设计**：每张模板卡包含模板名称、Prompt 内容摘要、一键复制按钮。

---

### 3.6 参数说明（Params）

**目的**：服务专业开发者，同时不吓到新手。

**布局**：两列，左侧参数说明列表 + 右侧工作台深色预览界面。

**专业参数**（面向游戏开发者）：
- Aspect Ratio — 画面比例（立绘推荐 1:1，场景推荐 16:9）
- Steps — 推理步数（质量/速度平衡）
- CFG Scale — Prompt 遵循度
- Seed — 固定种子复现结果

---

### 3.7 用户评价（Reviews）

**目的**：社会证明，针对游戏开发者的真实反馈。

**评价者画像**：优先展示独立游戏开发者、小型工作室创始人、游戏美术师。

---

### 3.8 相关工具推荐（Related Tools）

**目的**：延长用户停留，引导探索其他工具，提升工具组合使用率。

**推荐逻辑**（按工具关联性排序）：
- 角色立绘 → 细节精修（修手/脸）、动画预览（做动作验证）
- 场景原画 → 材质纹理（场景升频）
- Sprite 批量 → 细节精修（图标修图）
- 动画预览 → 角色立绘（上游输入）、细节精修（局部修改）

---

### 3.9 CTA Banner

**文案调整**（面向游戏开发者）：
- 标题："一个人，也能做出专业级游戏资产"
- 副文案："注册即送 30 积分，无需绑定信用卡。今天就生成你的第一款 AI 游戏角色。"

## 四、响应式策略

| 断点 | 变化 |
|------|------|
| > 1024px | 完整两列布局，4列瀑布流 |
