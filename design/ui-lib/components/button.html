Button 按钮

不是平面色块，而是有厚度、有材质的实体。每个状态过渡模拟物理运动 — 光线扩散、材质按压、表面折射。

与 v2 的根本区别：去掉 conic-gradient 旋转边框和所有装饰性动画。 Primary 是"发光体"而非"渐变色块" — 内发光 → hover 外扩散 → active 压回。 Secondary/Ghost 是"磨砂玻璃" — 半透明材质而非平面描边。 Active 统一用 scale(0.985) + 内阴影加深模拟"按压进材质"。
