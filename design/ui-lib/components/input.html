Input 输入框

不是"带边框的方框"，而是嵌入桌面的控制面板。有深度、有材质、有温度。聚焦时面板被点亮，像工作室的聚光灯打在你的工具上。

与 v2 的根本区别：不是 border + box-shadow 的标准组合。 输入框有内凹感（inset shadow），hover 时表面微亮（像触控板感应到手指）， focus 时品牌色光环从边缘自然扩散（三层 box-shadow 渐变过渡）。 新增浮动标签（floating label）和 Prompt 创作画布模式。
